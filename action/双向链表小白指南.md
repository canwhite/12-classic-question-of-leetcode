# 双向链表小白指南

## 🎯 什么是双向链表？

### 基本概念
**双向链表**是每个节点都有**两个指针**的链表：一个指向前一个节点，一个指向后一个节点。

### 通俗理解
```
单向链表就像单行道：
🏠 → 🏪 → 🏫 → 🏦
只能从前向后走

双向链表就像双向车道：
🏠 ↔️ 🏪 ↔️ 🏫 ↔️ 🏦
可以前后自由行走
```

### 链表结构对比
```
单向链表节点：
[数据|next] → [数据|next] → nil

双向链表节点：
nil ← [prev|数据|next] ↔️ [prev|数据|next] ↔️ [prev|数据|next] → nil
```

### 双向链表的基本结构
```go
// 双向链表节点
type DoublyNode struct {
    Val  int          // 数据：存储的值
    Prev *DoublyNode  // 前指针：指向前一个节点
    Next *DoublyNode  // 后指针：指向后一个节点
}

// 双向链表
type DoublyLinkedList struct {
    Head *DoublyNode  // 头节点
    Tail *DoublyNode  // 尾节点
    Size int          // 长度
}
```

## 🔥 双向链表的五大特征

### 1. **双向性（Two-way）**
- 可以向前遍历（通过Next指针）
- 可以向后遍历（通过Prev指针）
- 从任意节点可以到达其他任意节点
- 像双向马路，来回都畅通

### 2. **双端访问（Double-ended）**
- 有头节点和尾节点
- 可以从头部插入/删除
- 可以从尾部插入/删除
- 像地铁的两端都可以上下车

### 3. **自包含性（Self-contained）**
- 每个节点包含足够的信息
- 不需要从头节点开始查找前驱
- 删除节点更简单
- 每个节点都是一个小世界

### 4. **灵活操作（Flexible）**
- 插入删除都是O(1)（已知位置）
- 支持各种遍历方式
- 可以快速找到前后节点
- 操作起来很顺手

### 5. **额外开销（Extra overhead）**
- 每个节点多一个指针
- 内存占用稍大
- 代码逻辑稍复杂
- 但是换来更好的性能

## 🎮 双向链表解决的经典问题

### 1. **需要双向遍历的场景**
- 音乐播放器上一首/下一首
- 浏览器前进/后退功能
- 文本编辑器的光标移动
- 图片轮播的左右切换

### 2. **需要频繁头尾操作**
- 双端队列的实现
- LRU缓存系统
- 撤销/重做功能
- 任务调度系统

### 3. **需要快速删除节点**
- 操作系统的进程管理
- 数据库的事务处理
- 游戏中的对象管理
- 缓存替换算法

## 💡 双向链表的详细案例

### 案例1：创建和遍历双向链表
```go
package main

import "fmt"

func createDoublyList(values []int) *DoublyLinkedList {
    dll := &DoublyLinkedList{Head: nil, Tail: nil, Size: 0}

    for _, val := range values {
        dll.Append(val)
    }

    return dll
}

// 在尾部添加节点
func (dll *DoublyLinkedList) Append(val int) {
    newNode := &DoublyNode{Val: val, Prev: nil, Next: nil}

    if dll.Tail == nil {
        // 空链表
        dll.Head = newNode
        dll.Tail = newNode
    } else {
        // 链表不为空，连接到尾部
        dll.Tail.Next = newNode
        newNode.Prev = dll.Tail
        dll.Tail = newNode
    }

    dll.Size++
}

// 从前向后遍历
func (dll *DoublyLinkedList) ForwardPrint() {
    fmt.Print("前向遍历: ")
    current := dll.Head
    for current != nil {
        fmt.Printf("%d", current.Val)
        if current.Next != nil {
            fmt.Printf(" ↔️ ")
        }
        current = current.Next
    }
    fmt.Println(" → nil")
}

// 从后向前遍历
func (dll *DoublyLinkedList) BackwardPrint() {
    fmt.Print("后向遍历: ")
    current := dll.Tail
    for current != nil {
        fmt.Printf("%d", current.Val)
        if current.Prev != nil {
            fmt.Printf(" ↔️ ")
        }
        current = current.Prev
    }
    fmt.Println(" → nil")
}

func main() {
    dll := createDoublyList([]int{1, 2, 3, 4, 5})

    dll.ForwardPrint()
    dll.BackwardPrint()

    fmt.Printf("链表长度: %d\n", dll.Size)
    fmt.Printf("头节点值: %d\n", dll.Head.Val)
    fmt.Printf("尾节点值: %d\n", dll.Tail.Val)
}

// 运行结果：
// 前向遍历: 1 ↔️ 2 ↔️ 3 ↔️ 4 ↔️ 5 → nil
// 后向遍历: 5 ↔️ 4 ↔️ 3 ↔️ 2 ↔️ 1 → nil
// 链表长度: 5
// 头节点值: 1
// 尾节点值: 5
```

### 案例3：删除任意节点
```go
// 删除指定值的节点
func (dll *DoublyLinkedList) Delete(val int) {
    current := dll.Head

    for current != nil {
        if current.Val == val {
            if current.Prev != nil {
                current.Prev.Next = current.Next
            } else {
                // 删除的是头节点
                dll.Head = current.Next
            }

            if current.Next != nil {
                current.Next.Prev = current.Prev
            } else {
                // 删除的是尾节点
                dll.Tail = current.Prev
            }

            dll.Size--
            fmt.Printf("成功删除节点 %d\n", val)
            return
        }
        current = current.Next
    }

    fmt.Printf("未找到值为 %d 的节点\n", val)
}

// 删除头节点
func (dll *DoublyLinkedList) DeleteHead() {
    if dll.Head == nil {
        return
    }

    if dll.Head.Next == nil {
        // 只有一个节点
        dll.Head = nil
        dll.Tail = nil
    } else {
        // 有多个节点
        dll.Head = dll.Head.Next
        dll.Head.Prev = nil
    }

    dll.Size--
}

// 删除尾节点
func (dll *DoublyLinkedList) DeleteTail() {
    if dll.Tail == nil {
        return
    }

    if dll.Tail.Prev == nil {
        // 只有一个节点
        dll.Head = nil
        dll.Tail = nil
    } else {
        // 有多个节点
        dll.Tail = dll.Tail.Prev
        dll.Tail.Next = nil
    }

    dll.Size--
}

func main() {
    dll := createDoublyList([]int{1, 2, 3, 4, 5})
    fmt.Println("原始链表:")
    dll.ForwardPrint()

    fmt.Println("\n删除节点 3:")
    dll.Delete(3)
    dll.ForwardPrint()

    fmt.Println("\n删除头节点:")
    dll.DeleteHead()
    dll.ForwardPrint()

    fmt.Println("\n删除尾节点:")
    dll.DeleteTail()
    dll.ForwardPrint()

    fmt.Println("\n尝试删除不存在的节点 99:")
    dll.Delete(99)
    dll.ForwardPrint()
}

// 运行结果：
// 原始链表:
// 前向遍历: 1 ↔️ 2 ↔️ 3 ↔️ 4 ↔️ 5 → nil
//
// 删除节点 3:
// 成功删除节点 3
// 前向遍历: 1 ↔️ 2 ↔️ 4 ↔️ 5 → nil
//
// 删除头节点:
// 前向遍历: 2 ↔️ 4 ↔️ 5 → nil
//
// 删除尾节点:
// 前向遍历: 2 ↔️ 4 → nil
//
// 尝试删除不存在的节点 99:
// 未找到值为 99 的节点
// 前向遍历: 2 ↔️ 4 → nil
```

### 案例4：双向链表实现LRU缓存
```go
// LRU缓存节点
type LRUNode struct {
    Key   int
    Value int
    Prev  *LRUNode
    Next  *LRUNode
}

// LRU缓存
type LRUCache struct {
    capacity int
    size     int
    cache    map[int]*LRUNode
    head     *LRUNode
    tail     *LRUNode
}

// 创建LRU缓存
func Constructor(capacity int) *LRUCache {
    lru := &LRUCache{
        capacity: capacity,
        size:     0,
        cache:    make(map[int]*LRUNode),
        head:     &LRUNode{}, // 哑节点
        tail:     &LRUNode{}, // 哑节点
    }

    lru.head.Next = lru.tail
    lru.tail.Prev = lru.head
    return lru
}

// 移动节点到头部
func (lru *LRUCache) moveToHead(node *LRUNode) {
    lru.removeNode(node)
    lru.addToHead(node)
}

// 添加节点到头部
func (lru *LRUCache) addToHead(node *LRUNode) {
    node.Prev = lru.head
    node.Next = lru.head.Next
    lru.head.Next.Prev = node
    lru.head.Next = node
}

// 移除节点
func (lru *LRUCache) removeNode(node *LRUNode) {
    node.Prev.Next = node.Next
    node.Next.Prev = node.Prev
}

// 移除尾部节点
func (lru *LRUCache) removeTail() *LRUNode {
    lastNode := lru.tail.Prev
    lru.removeNode(lastNode)
    return lastNode
}

// 获取缓存值
func (lru *LRUCache) Get(key int) int {
    if node, exists := lru.cache[key]; exists {
        // 移动到头部（最近使用）
        lru.moveToHead(node)
        fmt.Printf("缓存命中: key=%d, value=%d (移动到头部)\n", key, node.Value)
        return node.Value
    }

    fmt.Printf("缓存未命中: key=%d\n", key)
    return -1
}

// 设置缓存值
func (lru *LRUCache) Put(key int, value int) {
    if node, exists := lru.cache[key]; exists {
        // 更新值并移动到头部
        node.Value = value
        lru.moveToHead(node)
        fmt.Printf("更新缓存: key=%d, value=%d (移动到头部)\n", key, value)
    } else {
        // 创建新节点
        newNode := &LRUNode{Key: key, Value: value}
        lru.cache[key] = newNode
        lru.addToHead(newNode)
        lru.size++

        fmt.Printf("添加缓存: key=%d, value=%d\n", key, value)

        // 检查容量
        if lru.size > lru.capacity {
            // 移除最久未使用的节点
            tail := lru.removeTail()
            delete(lru.cache, tail.Key)
            lru.size--
            fmt.Printf("缓存已满，移除最久未使用的: key=%d, value=%d\n", tail.Key, tail.Value)
        }
    }
}

func main() {
    cache := Constructor(3)

    fmt.Println("LRU缓存测试 (容量=3):")

    cache.Put(1, 100)
    cache.Put(2, 200)
    cache.Put(3, 300)

    fmt.Println("\n访问各个缓存:")
    cache.Get(1)  // 命中
    cache.Get(4)  // 未命中

    fmt.Println("\n添加新缓存 (会触发LRU淘汰):")
    cache.Put(4, 400)  // 容量已满，移除key=2

    fmt.Println("\n访问缓存:")
    cache.Get(1)  // 应该还在
    cache.Get(2)  // 应该被移除了
    cache.Get(3)
    cache.Get(4)
}

// 运行结果：
// LRU缓存测试 (容量=3):
// 添加缓存: key=1, value=100
// 添加缓存: key=2, value=200
// 添加缓存: key=3, value=300
//
// 访问各个缓存:
// 缓存命中: key=1, value=100 (移动到头部)
// 缓存未命中: key=4
//
// 添加新缓存 (会触发LRU淘汰):
// 添加缓存: key=4, value=400
// 缓存已满，移除最久未使用的: key=2, value=200
//
// 访问缓存:
// 缓存命中: key=1, value=100 (移动到头部)
// 缓存未命中: key=2
// 缓存命中: key=3, value=300 (移动到头部)
// 缓存命中: key=4, value=400 (移动到头部)
```

```go
    if mp.tail == nil {
        // 第一首歌
        mp.head = newSong
        mp.tail = newSong
        mp.current = newSong

        mp.current = mp.current.Next

        fmt.Println("已经是最后一首歌了")
```

## 🌳 双向链表在实际应用中的优势

### 1. **内存管理**
```go
// 操作系统中的进程管理
type Process struct {
    PID     int
    Name    string
    Status  string
    Prev    *Process
    Next    *Process
}

// 可以快速删除任意进程
func removeProcess(head *Process, pid int) *Process {
    current := head

    for current != nil {
        if current.PID == pid {
            if current.Prev != nil {
                current.Prev.Next = current.Next
            }
            if current.Next != nil {
                current.Next.Prev = current.Prev
            }
            break
        }
        current = current.Next
    }

    return head
}
```

### 2. **文本编辑器**
```go
// 文本编辑器的光标移动
type TextNode struct {
    Char rune
    Prev *TextNode
    Next *TextNode
}

type TextEditor struct {
    cursor *TextNode
    head   *TextNode
    tail   *TextNode
}

func (te *TextEditor) MoveLeft() {
    if te.cursor.Prev != nil {
        te.cursor = te.cursor.Prev
    }
}

func (te *TextEditor) MoveRight() {
    if te.cursor.Next != nil {
        te.cursor = te.cursor.Next
    }
}

func (te *TextEditor) DeleteAtCursor() {
    if te.cursor == nil {
        return
    }

    toDelete := te.cursor

    if toDelete.Prev != nil {
        toDelete.Prev.Next = toDelete.Next
    } else {
        te.head = toDelete.Next
    }

    if toDelete.Next != nil {
        toDelete.Next.Prev = toDelete.Prev
        te.cursor = toDelete.Next
    } else {
        te.tail = toDelete.Prev
        te.cursor = te.tail
    }
}
```

## ⚡ 双向链表 vs 其他数据结构

### 双向链表 vs 单向链表
```
双向链表优势：
✓ 可以双向遍历
✓ 删除节点更简单O(1)
✓ 支持双端操作
✓ 更灵活的操作

双向链表劣势：
✓ 每个节点多一个指针
✓ 内存占用稍大
✓ 插入操作稍复杂
```

## 🚀 双向链表的优化技巧

### 1. **使用哨兵节点**
```go
type DoublyLinkedList struct {
    head *DoublyNode  // 哨兵头节点
    tail *DoublyNode  // 哨兵尾节点
    size int
}

// 初始化时创建哨兵节点
func NewDoublyLinkedList() *DoublyLinkedList {
    head := &DoublyNode{}
    tail := &DoublyNode{}
    head.Next = tail
    tail.Prev = head
    return &DoublyLinkedList{head: head, tail: tail, size: 0}
}

// 操作时不需要检查nil
func (dll *DoublyLinkedList) InsertAfter(prev *DoublyNode, val int) *DoublyNode {
    newNode := &DoublyNode{Val: val}
    newNode.Prev = prev
    newNode.Next = prev.Next
    prev.Next.Prev = newNode
    prev.Next = newNode
    dll.size++
    return newNode
}
```

### 2. **批量操作优化**
```go
// 批量插入
func (dll *DoublyLinkedList) BatchInsert(values []int) {
    for _, val := range values {
        dll.Append(val)
    }
}

// 批量删除
func (dll *DoublyLinkedList) BatchDelete(values []int) {
    toDelete := make(map[int]bool)
    for _, val := range values {
        toDelete[val] = true
    }

    current := dll.Head
    for current != nil {
        next := current.Next
        if toDelete[current.Val] {
            dll.deleteNode(current)
        }
        current = next
    }
}
```

### 3. **内存池管理**
```go
// 节点池，减少内存分配
type NodePool struct {
    pool []*DoublyNode
}

func (np *NodePool) Get() *DoublyNode {
    if len(np.pool) > 0 {
        node := np.pool[len(np.pool)-1]
        np.pool = np.pool[:len(np.pool)-1]
        return node
    }
    return &DoublyNode{}
}

func (np *NodePool) Put(node *DoublyNode) {
    node.Prev = nil
    node.Next = nil
    np.pool = append(np.pool, node)
}
```

### 2. **验证链表完整性**
```go
func (dll *DoublyLinkedList) Validate() bool {
    if dll.Head == nil && dll.Tail == nil {
        return dll.Size == 0
    }

    // 检查从头到尾的连接
    forwardCount := 0
    current := dll.Head
    for current != nil {
        forwardCount++
        current = current.Next
    }

    // 检查从尾到头的连接
    backwardCount := 0
    current = dll.Tail
    for current != nil {
        backwardCount++
        current = current.Prev
    }

    return forwardCount == backwardCount && forwardCount == dll.Size
}
```

```
老后节点.prev = 新节点

删除节点：
老前节点.next = 老后节点
老后节点.prev = 老前节点
```
