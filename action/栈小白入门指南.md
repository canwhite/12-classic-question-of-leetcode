# 栈小白入门指南

## 🎯 什么是栈？超简单理解！

### 用生活例子理解栈
想象你有一叠盘子：
```
🔴🟢🔵  (最上面是红色盘子)

放盘子：只能在最上面放
🟡🔴🟢🔵  (黄色盘子放在最上面)

取盘子：只能从最上面取
🔴🟢🔵  (取走了黄色盘子)
```

### 核心规则：后进先出
- **后放的先取出来**
- **先放的后取出来**
- 只能在最上面操作
- 不能从中间或下面拿

---

## 🔥 栈的三大特征

### 1. **只能从顶部操作**
```
❌ 不能从中间插入
❌ 不能从下面取出
✅ 只能在最上面放东西
✅ 只能在最上面拿东西
```

### 2. **后进先出（LIFO）**
```
就像穿衣服：
先穿内衣，再穿衬衫，最后穿外套

脱衣服：
先脱外套，再脱衬衫，最后脱内衣
```

### 3. **操作很简单**
```
就三个基本动作：
push  → 放东西进去
pop   → 拿东西出来
peek  → 看看最上面是什么
```

## 🎮 生活中的栈应用

### 1. **浏览器后退按钮**
```
你访问网页的顺序：
A → B → C → D

按后退按钮的顺序：
D → C → B → A

就像栈一样，最后访问的网页先退出来！
```

### 2. **撤销功能**
```
你在编辑器里的操作：
打字 → 插入图片 → 改颜色 → 删除文字

撤销的顺序：
删除文字 → 改颜色 → 插入图片 → 打字

最后做的操作先撤销！
```

### 3. **一摞书**
```
放书顺序：数学 → 语文 → 英语 → 历史
(数学在最下面，历史在最上面)

取书顺序：历史 → 英语 → 语文 → 数学
(先取最上面的历史书)
```

## 💡 超简单的栈案例

### 案例1：理解基本操作
```go
package main

import "fmt"

func main() {
    // 用切片模拟栈
    var stack []string

    fmt.Println("空栈:", stack)

    // push：入栈（放东西）
    stack = append(stack, "苹果")
    fmt.Println("放入苹果:", stack)

    stack = append(stack, "香蕉")
    fmt.Println("放入香蕉:", stack)

    stack = append(stack, "橙子")
    fmt.Println("放入橙子:", stack)

    // pop：出栈（拿东西）
    if len(stack) > 0 {
        last := stack[len(stack)-1]  // 取最后一个
        stack = stack[:len(stack)-1] // 移除最后一个
        fmt.Println("拿出了:", last)
        fmt.Println("现在栈里有:", stack)
    }

    // peek：查看最上面
    if len(stack) > 0 {
        top := stack[len(stack)-1]
        fmt.Println("最上面是:", top)
        fmt.Println("栈里面还有:", stack)
    }
}

// 运行结果：
// 空栈: []
// 放入苹果: [苹果]
// 放入香蕉: [苹果 香蕉]
// 放入橙子: [苹果 香蕉 橙子]
// 拿出了: 橙子
// 现在栈里有: [苹果 香蕉]
// 最上面是: 香蕉
// 栈里面还有: [苹果 香蕉]
```

### 案例2：括号匹配（超级常用）
```go
package main

import "fmt"

func checkBrackets(s string) bool {
    var stack []rune

    // 括号配对表
    pairs := map[rune]rune{
        ')': '(',
        '}': '{',
        ']': '[',
    }

    for _, char := range s {
        // 如果是左括号，放入栈
        if char == '(' || char == '{' || char == '[' {
            stack = append(stack, char)
            fmt.Printf("放入左括号 %c，栈现在是: %c\n", char, stack)
        }
        // 如果是右括号，检查是否匹配
        else if char == ')' || char == '}' || char == ']' {
            if len(stack) == 0 {
                fmt.Printf("栈是空的，无法匹配 %c\n", char)
                return false
            }

            top := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            fmt.Printf("拿出左括号 %c 来匹配 %c，栈现在是: %c\n", top, char, stack)

            if pairs[char] != top {
                fmt.Printf("括号不匹配：%c 和 %c\n", top, char)
                return false
            }
        }
    }

    // 检查栈是否为空
    if len(stack) == 0 {
        fmt.Println("所有括号都匹配成功！")
        return true
    } else {
        fmt.Printf("栈里还有 %c，括号不匹配\n", stack)
        return false
    }
}

func main() {
    // 测试案例
    test1 := "({[]})"
    test2 := "({[})"

    fmt.Println("测试1:", test1)
    result1 := checkBrackets(test1)
    fmt.Println("结果:", result1)

    fmt.Println("\n测试2:", test2)
    result2 := checkBrackets(test2)
    fmt.Println("结果:", result2)
}

// 运行结果：
// 测试1: ({[]})
// 放入左括号 (，栈现在是: [(]
// 放入左括号 {，栈现在是: [({]
// 放入左括号 [，栈现在是: [{[]
// 拿出左括号 [ 来匹配 ]，栈现在是: [({]
// 拿出左括号 { 来匹配 }，栈现在是: [(]
// 拿出左括号 ( 来匹配 )，栈现在是: []
// 所有括号都匹配成功！
// 结果: true
```

### 案例3：数字反转
```go
package main

import "fmt"

func reverseNumber(n int) int {
    var stack []int

    // 把每一位数字放入栈
    for n > 0 {
        digit := n % 10      // 取最后一位
        stack = append(stack, digit)
        n = n / 10           // 去掉最后一位
        fmt.Printf("放入数字 %d，栈现在是: %v\n", digit, stack)
    }

    // 从栈中取出数字，组成反转后的数
    result := 0
    for len(stack) > 0 {
        last := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = result*10 + last
        fmt.Printf("取出数字 %d，当前结果: %d\n", last, result)
    }

    return result
}

func main() {
    num := 1234
    fmt.Printf("反转数字 %d\n", num)
    reversed := reverseNumber(num)
    fmt.Printf("反转结果: %d\n", reversed)
}

// 运行结果：
// 反转数字 1234
// 放入数字 4，栈现在是: [4]
// 放入数字 3，栈现在是: [4 3]
// 放入数字 2，栈现在是: [4 3 2]
// 放入数字 1，栈现在是: [4 3 2 1]
// 取出数字 1，当前结果: 1
// 取出数字 2，当前结果: 12
// 取出数字 3，当前结果: 123
// 取出数字 4，当前结果: 1234
// 反转结果: 4321
```

## 🔥 单调栈：栈的高级应用

### 什么是单调栈？
**单调栈**是栈里面元素保持单调性（单调递增或单调递减）的特殊栈。

### 通俗理解单调栈
想象你在排队，要求队伍必须按身高从矮到高排列：

```
正常排队：160, 165, 170, 168
插入168时：160, 165, [168]出来, 170, [168]进去
新队列：160, 165, 168, 170
```

### 单调递增栈（从底到顶递增）
```go
package main

import "fmt"

// 单调递增栈：栈底到栈顶元素递增
func monotonicIncreasingStack(nums []int) []int {
    var stack []int
    result := make([]int, len(nums))

    // 初始化所有位置为-1（表示右边没有更大的元素）
    for i := range result {
        result[i] = -1
    }

    fmt.Printf("原始数组: %v\n", nums)

    for i, num := range nums {
        fmt.Printf("\n处理数字 %d (位置%d):\n", num, i)

        // 维护单调递增：如果当前数字比栈顶大，弹出栈顶
        for len(stack) > 0 && nums[stack[len(stack)-1]] < num {
            top := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            result[top] = num
            fmt.Printf("  弹出位置 %d (值=%d)，因为 %d < %d\n",
                top, nums[top], nums[top], num)
            fmt.Printf("  位置 %d 的下一个更大元素是 %d\n", top, num)
            fmt.Printf("  当前栈: %v\n", stack)
        }

        // 当前位置入栈
        stack = append(stack, i)
        fmt.Printf("  位置 %d 入栈\n", i)
        fmt.Printf("  当前栈: %v\n", stack)
    }

    fmt.Printf("\n最终结果 (每个元素右边第一个更大的元素): %v\n", result)
    return result
}

func main() {
    nums := []int{2, 1, 2, 4, 3}
    fmt.Println("寻找每个元素右边第一个更大的元素")
    result := monotonicIncreasingStack(nums)
}

// 运行结果：
// 原始数组: [2 1 2 4 3]
// 寻找每个元素右边第一个更大的元素
//
// 处理数字 2 (位置0):
//   位置 0 入栈
//   当前栈: [0]
//
// 处理数字 1 (位置1):
//   位置 1 入栈
//   当前栈: [0 1]
//
// 处理数字 2 (位置2):
//   弹出位置 1 (值=1)，因为 1 < 2
//   位置 1 的下一个更大元素是 2
//   当前栈: [0]
//   位置 2 入栈
//   当前栈: [0 2]
//
// 处理数字 4 (位置3):
//   弹出位置 2 (值=2)，因为 2 < 4
//   位置 2 的下一个更大元素是 4
//   当前栈: [0]
//   弹出位置 0 (值=2)，因为 2 < 4
//   位置 0 的下一个更大元素是 4
//   当前栈: []
//   位置 3 入栈
//   当前栈: [3]
//
// 处理数字 3 (位置4):
//   位置 4 入栈
//   当前栈: [3 4]
//
// 最终结果 (每个元素右边第一个更大的元素): [4 2 4 -1 -1]
```

### 单调递减栈（从底到顶递减）
```go
package main

import "fmt"

// 单调递减栈：寻找左边第一个更大的元素
func monotonicDecreasingStack(nums []int) []int {
    var stack []int
    result := make([]int, len(nums))

    // 初始化所有位置为-1
    for i := range result {
        result[i] = -1
    }

    fmt.Printf("原始数组: %v\n", nums)

    for i, num := range nums {
        fmt.Printf("\n处理数字 %d (位置%d):\n", num, i)

        // 维护单调递减：如果当前数字比栈顶大，弹出栈顶
        for len(stack) > 0 && nums[stack[len(stack)-1]] <= num {
            top := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            // 注意：这里是当前数字的左边第一个更大元素
            result[i] = nums[top]
            fmt.Printf("  找到位置 %d 左边第一个更大的元素: %d\n", i, nums[top])
        }

        // 当前位置入栈
        stack = append(stack, i)
        fmt.Printf("  位置 %d 入栈\n", i)
        fmt.Printf("  当前栈: %v\n", stack)
    }

    fmt.Printf("\n最终结果 (每个元素左边第一个更大的元素): %v\n", result)
    return result
}

func main() {
    nums := []int{3, 1, 4, 2, 5}
    fmt.Println("寻找每个元素左边第一个更大的元素")
    result := monotonicDecreasingStack(nums)
}

// 运行结果：
// 原始数组: [3 1 4 2 5]
// 寻找每个元素左边第一个更大的元素
//
// 处理数字 3 (位置0):
//   位置 0 入栈
//   当前栈: [0]
//
// 处理数字 1 (位置1):
//   位置 1 入栈
//   当前栈: [0 1]
//
// 处理数字 4 (位置2):
//   弹出位置 1 (值=1)，因为 1 <= 4
//   弹出位置 0 (值=3)，因为 3 <= 4
//   位置 2 左边第一个更大的元素: 3
//   位置 2 入栈
//   当前栈: [2]
//
// 处理数字 2 (位置3):
//   位置 3 入栈
//   当前栈: [2 3]
//
// 处理数字 5 (位置4):
//   弹出位置 3 (值=2)，因为 2 <= 5
//   弹出位置 2 (值=4)，因为 4 <= 5
//   位置 4 左边第一个更大的元素: 4
//   位置 4 入栈
//   当前栈: [4]
//
// 最终结果 (每个元素左边第一个更大的元素): [-1 -1 3 4 4]
```

### 单调栈的实际应用案例

#### 案例1：每日温度
```go
// 问题：给定温度数组，求要等几天才能遇到更热的温度
func dailyTemperatures(temperatures []int) []int {
    n := len(temperatures)
    result := make([]int, n)
    var stack []int  // 存储位置

    for i := n - 1; i >= 0; i-- {
        // 弹出比当前温度小的位置
        for len(stack) > 0 && temperatures[stack[len(stack)-1]] <= temperatures[i] {
            stack = stack[:len(stack)-1]
        }

        // 如果栈不为空，计算天数差
        if len(stack) > 0 {
            result[i] = stack[len(stack)-1] - i
        } else {
            result[i] = 0
        }

        // 当前位置入栈
        stack = append(stack, i)
    }

    return result
}

func main() {
    temps := []int{73, 74, 75, 71, 69, 72, 76, 73}
    fmt.Printf("温度: %v\n", temps)
    result := dailyTemperatures(temps)
    fmt.Printf("等待天数: %v\n", result)
    // 结果: [1, 1, 4, 2, 1, 1, 0, 0]
    // 解释: 73°要等1天到74°，75°要等4天到76°，等等
}
```

#### 案例2：柱状图中的最大矩形
```go
// 问题：给定柱状图高度，求最大矩形面积
func largestRectangleArea(heights []int) int {
    // 在末尾加0，确保所有柱子都被处理
    heights = append(heights, 0)
    var stack []int  // 存储位置
    maxArea := 0

    fmt.Printf("柱状图高度: %v\n", heights)

    for i, height := range heights {
        fmt.Printf("\n处理位置 %d，高度 %d:\n", i, height)

        // 当前高度比栈顶低，弹出计算面积
        for len(stack) > 0 && heights[stack[len(stack)-1]] > height {
            h := heights[stack[len(stack)-1]]
            stack = stack[:len(stack)-1]

            var width int
            if len(stack) == 0 {
                width = i
            } else {
                width = i - stack[len(stack)-1] - 1
            }

            area := h * width
            maxArea = max(maxArea, area)
            fmt.Printf("  弹出高度 %d，宽度 %d，面积 %d\n", h, width, area)
            fmt.Printf("  当前最大面积: %d\n", maxArea)
        }

        stack = append(stack, i)
        fmt.Printf("  位置 %d 入栈\n", i)
    }

    return maxArea
}

func main() {
    heights := []int{2, 1, 5, 6, 2, 3}
    fmt.Printf("柱状图高度: %v\n", heights)
    area := largestRectangleArea(heights)
    fmt.Printf("最大矩形面积: %d\n", area)
    // 结果: 10 (由高度5和6的两根柱子组成)
}
```

#### 案例3：滑动窗口最大值
```go
// 问题：给定数组和窗口大小，求每个滑动窗口的最大值
func maxSlidingWindow(nums []int, k int) []int {
    if len(nums) == 0 || k == 0 {
        return []int{}
    }

    var deque []int  // 存储位置，保持递减
    result := []int{}

    for i, num := range nums {
        // 移除窗口外的元素
        for len(deque) > 0 && deque[0] <= i-k {
            deque = deque[1:]
        }

        // 保持递减，移除比当前元素小的
        for len(deque) > 0 && nums[deque[len(deque)-1]] < num {
            deque = deque[:len(deque)-1]
        }

        deque = append(deque, i)

        // 窗口形成后，记录最大值
        if i >= k-1 {
            result = append(result, nums[deque[0]])
        }
    }

    return result
}

func main() {
    nums := []int{1, 3, -1, -3, 5, 3, 6, 7}
    k := 3
    fmt.Printf("数组: %v, 窗口大小: %d\n", nums, k)
    result := maxSlidingWindow(nums, k)
    fmt.Printf("滑动窗口最大值: %v\n", result)
    // 结果: [3, 3, 5, 5, 6, 7]
    // 窗口: [1,3,-1] [3,-1,-3] [-1,-3,5] [-3,5,3] [5,3,6] [3,6,7]
}
```

## 🎮 单调栈的核心思想

### 为什么叫"单调"？
```
单调递增栈：从栈底到栈顶，元素越来越大
            [1, 3, 5, 7, 9]

单调递减栈：从栈底到栈顶，元素越来越小
            [9, 7, 5, 3, 1]
```

### 单调栈的通用模式
```go
// 通用模板
func monotonicStackPattern(nums []int) {
    var stack []int

    for i, num := range nums {
        // 维护单调性的关键代码
        for len(stack) > 0 && nums[stack[len(stack)-1]] < num {
            top := stack[len(stack)-1]
            stack = stack[:len(stack)-1]

            // 在这里处理弹出的元素
            // 比如：计算面积、找到下一个更大元素等
            processElement(top, i, num)
        }

        // 当前元素入栈
        stack = append(stack, i)
    }

    // 处理栈中剩余元素
    for len(stack) > 0 {
        top := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        processRemainingElement(top)
    }
}
```

### 单调栈解决什么问题？
```
1. 寻找下一个更大/更小的元素
2. 寻找左边/右边第一个满足条件的元素
3. 计算矩形面积、最大值等
4. 滑动窗口最值问题
```

## 🎯 单调栈记忆技巧

### 生活类比
```
单调递增栈：
    就像排队，矮的在前，高的在后
    新来的人比前面高，前面的人要让位

单调递减栈：
    就像叠盘子，大盘子在下，小盘子在上
    放小盘子时，大一点的盘子要拿走
```

### 判断使用哪种单调栈
```
问题中出现：
✓ "下一个更大/更高/更热" → 单调递增栈
✓ "左边第一个更大的" → 单调递减栈
✓ "最大矩形面积" → 单调递增栈
✓ "滑动窗口最大值" → 单调递减队列（类似栈）
```

### 单调栈vs普通栈
```
普通栈：随意放入取出
        像普通的盘子堆

单调栈：有规则的放入取出
        像按身高排队
        维护特定的顺序
```

## 🎯 栈vs其他数据结构

### 栈 vs 队列
```
栈：后进先出
    像子弹夹，后装的子弹先发射
    🔄 [1,2,3] → 放4 → [1,2,3,4] → 取出 → 4

队列：先进先出
    像排队，先来的人先服务
    🔄 [1,2,3] → 加入4 → [1,2,3,4] → 取出 → 1
```

### 栈 vs 数组
```
栈：只能在顶部操作
    像一摞盘子，只能拿最上面的
    操作快，但功能有限

数组：可以随意访问
    像书架，可以拿任意位置的书
    功能多，但插入删除慢
```
